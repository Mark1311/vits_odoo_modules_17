Advance Hide Tracking Number

Hide tracking numbers on POS receipts instantly with a single backend toggle.

The Advance Hide Tracking Number module for Odoo 18 enables businesses to seamlessly manage the display of tracking numbers on POS receipts. With a simple backend toggle in the POS settings, users can choose to hide or display tracking numbers instantly, ensuring clean and professional-looking receipts. Fully integrated without modifying core templates, this lightweight module provides a flexible, user-friendly solution for enhancing the POS experience and maintaining operational clarity.

Advance Hide Tracking Number Workflow

  • Install the POS Hide Tracking Number module in Odoo.
  • Go to POS Settings and locate the POS Hide Tracking Number on Receipt section.
  • Enable the Hide Tracking Number toggle to hide tracking numbers on POS receipts.
  • In the same section, enable the Hide Powered by Odoo toggle to remove the ‚ÄúPowered by Odoo‚Äù text from the bottom of POS receipts. Disable it if you want to show the branding text again.
  • Start a POS session from the POS dashboard.
  • When the Hide Tracking Number option is enabled, all POS receipts will automatically hide tracking numbers. If disabled, tracking numbers will be displayed normally.
  • When the Hide Powered by Odoo option is enabled, the branding text will be hidden; if disabled, it will be visible.
  • Continue processing POS orders as usual, with receipt display fully controlled by these backend settings.
