Remove/Hide Powered by Odoo Footer

Take full control of your site’s branding: remove the ‘Powered by Odoo’ footer with a single switch.

Remove/Hide Powered by Odoo Footer is a lightweight module that lets you easily remove the default “Powered by Odoo” branding from your website and login pages. With a simple backend toggle in POS configuration, you can instantly hide the footer text, giving your platform a clean and professional look without modifying core files.

Remove/Hide Powered by Odoo Footer Workflow

  • Install the Remove/Hide Powered by Odoo Footer module in Odoo.
  • Once installed, the module automatically removes the “Powered by Odoo” text from the website home page and login page.
  • Refresh your website or login page to see the changes.
  • When the module is active, the footer branding will no longer be displayed. If the module is uninstalled or disabled, the “Powered by Odoo” text will reappear automatically.
  • Continue using your website and login pages as usual, with footer branding fully managed by this module.
