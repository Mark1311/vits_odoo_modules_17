Restrict POS Invoice Download Workflow

  • Install the Restrict POS Invoice Download | Invoice Automation PRO module in your Odoo system.
  • Go to Point of Sale > Settings and scroll to the POS Auto Invoice Generate section.
  • Enable the "Enable Auto Invoice Generate" option to automatically create invoices after each order.
  • Once enabled, a second option "Disable Auto Invoice PDF Download" will appear.
  • Enable this option if you want to stop automatic invoice PDF downloads after each payment.
  • Save the settings to apply your configuration.
  • Start a new POS session from the POS dashboard.
  • Create a new order, proceed to payment, and validate the order as usual.
  • The invoice will be automatically generated in the backend.
  • If PDF download is not disabled, the invoice PDF will also be downloaded automatically.
  • If PDF download is disabled, only the invoice will be created without downloading the PDF.
  • All invoices remain accessible from the backend via Point of Sale > Orders > Invoices tab.
