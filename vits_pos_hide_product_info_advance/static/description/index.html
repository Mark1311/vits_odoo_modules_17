The POS Hide Product Info Advance module gives you complete control over product information visibility inside Odoo POS. Instead of hiding the entire product info panel, this advanced version allows you to selectively hide specific sections such as Header, Inventory, Replenishment, Financial details, Orders, and the Edit button. With flexible configuration options in POS Settings, businesses can optimize the interface based on user roles, reduce screen clutter, enhance security, and create a cleaner, distraction-free POS environment tailored to operational needs.

  • Go to Point of Sale > Settings and select the POS shop you want to configure.
  • Locate the ‚ÄúPOS Hide Product Info‚Äù section in the configuration panel.
  • Enable the specific fields (Header, Inventory, Replenishment, Financial, Orders, Edit Button) you want to hide from the product info panel.
